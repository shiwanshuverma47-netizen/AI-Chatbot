# MBA Buddy

A polished MBA study assistant chatbot UI with subject-based shortcuts, quick prompts, saved chat history, and a more professional assistant experience.

## Features
- Responsive chat layout
- Subject-based topic shortcuts
- Quick prompt chips for faster study questions
- Local autosave so conversations persist in the browser
- Assistant branding in the header and bottom-right corner

## Run locally
Open index.html in your browser or serve the folder with any static file server.
