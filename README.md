# MBA Buddy

A polished MBA study assistant chatbot UI with subject-based shortcuts, quick prompts, saved chat history, and a more professional assistant experience.

## ✨ Features

* 🎓 AI-powered chatbot for MBA-related queries
* 💬 Natural language conversations with contextual responses
* 📚 Guidance on MBA admissions, entrance exams, and career paths
* 🏫 Information about MBA specializations and colleges
* 📈 Career advice based on user interests
* ⚡ Fast and responsive chat interface
* 🛡️ Input validation and graceful error handling
* 📱 Clean, user-friendly, and responsive design


## 📁 Project Structure

```text
mba-buddy-ai-chatbot/
├── app.py                 # Main application entry point
├── requirements.txt       # Python dependencies
├── .env.example           # Environment variables template
├── assets/                # Images, icons, and static resources
├── src/                   # Application source code
│   ├── chatbot/           # Chatbot logic
│   ├── services/          # AI and API integrations
│   ├── utils/             # Helper functions
│   └── config/            # Configuration files
├── README.md              # Project documentation
└── LICENSE                # License information
```

### Folder Overview

* **app.py** – Starts the chatbot application.
* **src/chatbot** – Contains conversation handling and response generation.
* **src/services** – Manages AI model and external API communication.
* **src/utils** – Includes reusable helper functions.
* **assets** – Stores images and other static resources.
* **requirements.txt** – Lists all required Python packages.
