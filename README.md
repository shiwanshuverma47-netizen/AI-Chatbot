# MBA Buddy

A polished MBA study assistant chatbot UI with subject-based shortcuts, quick prompts, saved chat history, and a more professional assistant experience.

## ✨ Features

* 🎓 AI-powered chatbot for MBA-related queries
* 💬 Natural language conversations with contextual responses
* 📚 Guidance on MBA admissions, entrance exams, and career paths
* 🏫 Information about MBA specializations and colleges
* 📈 Career advice based on user interests
* ⚡ Fast and responsive chat interface
* 🛡️ Input validation and graceful error handling
* 📱 Clean, user-friendly, and responsive design


## Run locally
Open index.html in your browser or serve the folder with any static file server.
